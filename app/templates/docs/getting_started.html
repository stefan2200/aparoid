Getting started

Everything you need to get Aparoid set-up!

Installation

You can use this to set-up Aparoid:

  • Clone the Aparoid repository using 
    git clone https://github.com/stefan2200/aparoid
  • cd to the directory and run the setup script 
    cd aparoid; ./setup.sh
  • Start Aparoid by running 
    ./start.sh
  • Windows users have to run the setup.bat and start.bat scripts

Docker

You can also install Aparoid using docker:

  • Clone the Aparoid repository using 
    git clone https://github.com/stefan2200/aparoid
  • Make sure the docker engine is installed and working
  • cd to the directory and run 
    docker compose up -d

Optional utilities

The HTTP(S) proxy uses Kafka to process data asynchronously. If you have no problems reading the traffic from the terminal you can disable Kafka in the configuration file. I do however recommend installing Kafka either native or using Docker. You can follow the guide below to set-up Kafka using Docker.

Linux
  • Install the Docker Engine by following the steps on docker.com
  • Install Docker Compose by following the steps on docker.com
  • cd into the collector directory
  • Start the Kafka server by running 
    sudo docker-compose up -d
Windows
  • Install the Docker Desktop by following the steps on docker.com
  • cd into the collector directory
  • Start the Kafka server by running 
    docker-compose up -d

Updating

Aparoid will automatically update when running the setup script.

  • cd to the directory and run the setup script 
    cd aparoid; ./setup.sh
  • Windows users have to run the setup.bat script
