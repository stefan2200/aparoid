Introduction

What is Aparoid?

Aparoid is a framework for dynamic and static application analysis. It is built on top of the Flask framework and supports a wide variety of devices and Android frameworks.

Features

Some of the features that Aparoid static code analysis offers:

  • Automated code decompilation using jadx
  • Search patterns for common static code weaknesses, vulnerabilities and other dangers
  • Binary security analysis
  • Various checks for commonly used frameworks including: Flutter, Xamarin and React Native
  • Easy to use file browser

Some of the features that Aparoid dynamic analysis offers:

  • Customizable code hooks using frida
  • Custom built HTTP(S) interception proxy (including real-time traffic viewer)
  • Real-time application file browser
  • Basic security checks for HTTP traffic
  • Standard frida scripts to defeat root detection, SSL pinning and debugger detection
